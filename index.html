Stack Technique Proposé

🔧 Backend

  • Python 3.9+ - Langage principal
  • FastAPI - API REST (déjà en place)
  • Celery + Redis - Tâches async
  • PostgreSQL - Métadonnées

📄 Parsing

  • LangChain - Framework principal
  • Unstructured.io - Multi-format
  • PyPDF2 - PDFs complexes
  • Pandas - Excel/CSV

🎨 Frontend

  • Streamlit - MVP rapide
  • React - Version production
  • Tailwind CSS - Styling
  • Recharts - Visualisations

🧠 IA/Vectorisation

  • Pinecone - Base vectorielle
  • OpenAI - Embeddings
  • Sentence-Transformers - Alternative OS
  • GPT-4 - Agent conversationnel
